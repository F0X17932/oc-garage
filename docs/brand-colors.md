# OC Garage — Identidad de Color

## Color Principal (Acento)

| Formato | Valor |
|---------|-------|
| **HEX** | `#98EF04` |
| **RGB** | `152, 239, 4` |
| **CMYK** | `36, 0, 98, 6` |
| **Nombre** | Verde lima neón / Acid green |

**Origen:** Color extraído del logo oficial OC Garage (texto "OC" y elementos gráficos).  
**Uso:** Botones, acentos de texto, bordes, gradientes, cursor personalizado, ticker.
